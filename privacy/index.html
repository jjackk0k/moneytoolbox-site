Privacy

Short version: the calculators run entirely in your browser, and nothing you type leaves it.

What we collect

Nothing personal. There are no accounts, no signup, and no analytics or advertising trackers on the calculator pages. The numbers you enter (balances, rates, income) are processed by JavaScript on your own device and are never transmitted to or stored on a server.

Hosting

The site is served as static files by GitHub Pages, which — like any web host — may log basic request data (such as IP addresses) for security and operations. That's outside our control and standard for the web.

Links and offers

Some pages may eventually include clearly-labelled links to financial products. If a link is a paid/affiliate link, it will say so. We don't sell or share anything you enter, because we never receive it in the first place.
